# Support

We're here to help you adopt successfully.

## Email us

For anything not covered on this site (access issues, hub provisioning, Canvas import
errors, or grading questions), email **[{{ support_email }}](mailto:{{ support_email }})**.
When reporting an access problem, include the **GitHub username** you submitted on the
course's instructor interest form.

## Common issues

??? question "I can't access the private repositories"

    - Confirm you accepted all GitHub invitations (check the bell icon and your email spam
      folder).
    - Verify your GitHub username matches what you submitted on the interest form.
    - Open the solutions repo and the
      [otter-service-stdalone org]({{ otter_org_url }}) while logged in. GitHub often
      surfaces an "Accept" prompt there.
    - Still stuck after 24 hours? Email [{{ support_email }}](mailto:{{ support_email }}).

??? question "My JupyterHub isn't ready yet"

    If your hub is still being provisioned (for example through Cloudbank or Cal-ICOR),
    we'll give you a realistic timeline. For piloting, consider Google Colab or Binder in
    the meantime.

??? question "Canvas import errors"

    - Make sure you're importing a **Common cartridge 1.x package**.
    - Check Canvas file-size limits.
    - Try importing sections individually if the full import fails.
